# Prova SQL - Tabela Clientes

Este repositório contém um exemplo completo de SQL para a criação, inserção e manipulação da tabela **Clientes**.

 1. Criar a tabela

A tabela possui as colunas:

- `ID` → chave primária, autoincrementável
- `Nome` → nome do cliente
- `Idade` → idade do cliente
- `Cidade` → cidade do cliente

O comando para criar a tabela está no arquivo `tabela.sql`.

 2. Inserção de dados

O arquivo inclui exemplos de inserção de clientes:

- João, 30 anos, Fortaleza
- Maria, 25 anos, São Paulo
- Carlos, 40 anos, Rio de Janeiro
- Ana, 22 anos, Belo Horizonte

 3. Consultas e manipulação

O arquivo `tabela.sql` também contém:

- Seleção de todos os registros
- Seleção filtrada por cidade
- Atualização de dados
- Exclusão de registros

 4. Como usar

1. Instale um banco de dados (MySQL, MariaDB ou SQLite)
2. Abra o VS Code (ou outro cliente SQL)
3. Abra o arquivo `tabela.sql`
4. Execute todas as queries para criar e testar a tabela
